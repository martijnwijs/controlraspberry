def switch(data):

    # if toggle switch
    if data['type'] == "toggle":
        GPIO.setmode(GPIO.BCM)
        dictoutput={'LOW': GPIO.LOW, 'HIGH': GPIO.HIGH}
        GPIO.setup(dictpins[data['pin']],GPIO.OUT)
        GPIO.output(dictpins[data['pin']],dictoutput[data['value']])
        
    # if slider 
    if data['type'] == "slider":
        GPIO.setmode(GPIO.BCM)
        GPIO.setup(int([data['pin']), GPIO.OUT)
        pwm = GPIO.PWM(int(data['pin']),50)
        pwm.start(0)
        pwm.ChangeDutyCycle(int(data['value']))

        pwm.stop()
        GPIO.cleanup()

For controlling an actuator on the raspberry PI do the following:

  • 1. Make sure to connect your actuator to the raspberry PI in the right way.
  • 2. execute the code on your raspberry PI and copy the SID which will be printed in your terminal
  • 3. Add a controller in the app and link it to the same output pin as your actuator is attached to.
  • 4. paste the SID in your controller. now you should be able to control a actuator
  • 5. Add a controller in the app and link it to the same output pin as your actuator is attached to.


For executing code on the Raspberry PI, (or any other device), do the following:
  • 1. in the program on your raspberry PI, create a method.
  • 2. Create the code you want to execute inside this method. Because a toggle switch is used for the execution. you can split your code in two parts handling turning the toggle "ON" and one handles "OFF" as you like.
  • 3. Add a controller in the app and choose "execute code" for type. Make sure you give the controller the same name as the name of the method you have just written to execute code.
  • 4. paste the SID in your controller. now you should be able to execute the code.
